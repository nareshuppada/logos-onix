ONIX Code Lists Issue 20, January 2013
List 32: Complexity scheme identifier
Value Description Notes
01 Lexile code For example AD or HL. DEPRECATED in ONIX 3 – use <Audience> instead.
02 Lexile number For example 880L. DEPRECATED in ONIX 3 – use <Audience> instead.
03 Fry Readability score DEPRECATED in ONIX 3 – Use <Audience> instead. Fry readability metric based on number of sentences and syllables per 100 words. Expressed as a number from 1 to 15 in <ComplexityCode>.
